Welcome to Camus
================

Camus is a video chat app that uses `WebRTC`_ for direct peer-to-peer
communication. Users can create and join rooms, stream audio and video from
a webcam and microphone, share their screen, and send text messages.
You can try a demo at https://camus.chat, or download and run a server
yourself using :ref:`pip <Installation:Using Python>` or
:ref:`Docker <Installation:Using Docker>`.
